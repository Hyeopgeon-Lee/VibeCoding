# 학생 성적 분석 실습

`student.py`는 엑셀 파일에 저장된 학생 데이터를 읽어 성적 평균, 상관계수, 평균점수 상위 학생을 분석하고 산점도를 생성하는 실습 프로그램입니다.

## 실습 목표

- `pandas`를 사용하여 엑셀 데이터를 읽고 분석하기
- 영어점수와 수학점수의 평균을 계산하여 새 컬럼 추가하기
- 평균점수가 높은 학생 상위 3명 출력하기
- 공부시간과 평균점수의 관계를 산점도로 시각화하기
- 분석 결과 그래프를 이미지 파일로 저장하기

## 사용 파일

- `student.py`: 학생 데이터 분석 프로그램
- `data/student.xlsx`: 학생 원본 데이터 엑셀 파일
- `result/study_vs_avg_score.png`: 실행 후 생성되는 산점도 이미지 파일

## 주요 기능

1. 엑셀 파일 읽기
   - `data/student.xlsx` 파일을 읽어 학생 데이터를 불러옵니다.

2. 전체 데이터 출력
   - 학생ID, 이름, 영어점수, 수학점수, 주당평균공부시간, 주당평균수면시간, 평균점수를 출력합니다.

3. 평균점수 계산
   - 영어점수와 수학점수의 평균을 계산합니다.
   - 계산 결과를 `평균점수`라는 새 컬럼으로 추가합니다.

4. 평균값 분석
   - 영어점수 평균
   - 수학점수 평균
   - 주당평균공부시간 평균
   - 주당평균수면시간 평균

5. 평균점수 상위 3명 출력
   - 평균점수가 높은 순서대로 상위 3명을 출력합니다.
   - 출력 항목은 이름, 영어점수, 수학점수, 평균점수입니다.
   - 평균점수는 소수 둘째 자리까지 출력합니다.

6. 상관계수 분석
   - 영어점수와 주당평균공부시간의 상관계수
   - 수학점수와 주당평균공부시간의 상관계수
   - 영어점수와 주당평균수면시간의 상관계수
   - 수학점수와 주당평균수면시간의 상관계수

7. 산점도 생성
   - x축: `주당평균공부시간`
   - y축: `평균점수`
   - 그래프 제목: `Study Hours vs Average Score`
   - x축 이름: `Study Hours`
   - y축 이름: `Average Score`
   - 그래프를 화면에 출력합니다.
   - `result` 폴더를 자동으로 생성합니다.
   - 그래프를 `result/study_vs_avg_score.png` 파일로 저장합니다.

## 실행 방법

가상환경의 Python을 사용하여 실행합니다.

```powershell
.\.venv\Scripts\python.exe student.py
```

일반 Python 실행 환경에서는 다음 명령으로 실행할 수 있습니다.

```powershell
python student.py
```

## 필요 라이브러리

- pandas

## 실행 결과

프로그램을 실행하면 콘솔에 다음 분석 결과가 출력됩니다.

- 전체 학생 데이터
- 행 개수와 열 개수
- 주요 컬럼의 평균값
- 평균점수 상위 3명
- 점수와 공부시간, 수면시간의 상관계수

또한 산점도 그래프가 화면에 표시되고, 아래 경로에 이미지 파일이 저장됩니다.

```text
result/study_vs_avg_score.png
```
